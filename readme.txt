Displays an material design icon or material symbol, optionally as a link

== Description ==

Displays a icon from the library of Material Design Icons or the new Material Symbols.

All possible icon style customizations are available to the user. 

Allows the user to make the icon a link.

Allows several appearance customizations, including size, background shape, background color, icon color, and padding. 

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/icon-link` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
